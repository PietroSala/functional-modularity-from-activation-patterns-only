# Functional Modularity from Activation Patterns Only

This repository contains the code to reproduce the results presented in the paper:

- **Title:** Functional Modularity from Activation Patterns Only
- **Authors:** Cesare Montresor, Pietro Sala
- **Affiliation:** Department of Computer Science, University of Verona, Verona, Italy
- **Publication status:** Accepted as a regular paper at the upcoming conference [IEEE CAI 2026](https://www.ieeesmc.org/cai-2026/).
